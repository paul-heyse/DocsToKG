1. Module: io

This reference documents the DocsToKG module DocsToKG.DocParsing.io.

1. Overview

Low-level I/O helpers shared across DocParsing stages.

This module houses JSONL streaming utilities, atomic write helpers, and manifest bookkeeping routines. It deliberately avoids importing the CLI-facing modules so that other packages can depend on these primitives without pulling in heavy dependencies.
