1. Module: retrieval

This reference documents the DocsToKG module DocsToKG.HybridSearch.retrieval.

Deprecated: DocsToKG.HybridSearch.retrieval is now a thin shim that re-exports the public interface from DocsToKG.HybridSearch.service. Import :class:HybridSearchService, :class:HybridSearchAPI, and supporting helpers from the service module directly. The shim emits a :class:DeprecationWarning on import.
