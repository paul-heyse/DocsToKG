1. Module: env

This reference documents the DocsToKG module DocsToKG.DocParsing.env.

1. Overview

Environment and path helpers for DocParsing.

This module centralises filesystem discovery, environment initialisation, and dependency checks so that orchestrators can rely on a single location for these concerns. Importing the module is side-effect free; directories and environment variables are only modified when the exported functions are invoked.
