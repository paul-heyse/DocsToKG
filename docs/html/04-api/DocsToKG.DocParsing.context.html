1. Module: context

This reference documents the DocsToKG module DocsToKG.DocParsing.context.

1. Overview

Typed runtime context shared across DocParsing stages.

The :class:ParsingContext dataclass captures the run-scoped attributes that the CLI and pipeline stages exchange. It replaces loosely typed dictionaries so callers benefit from IDE completion, static analysis, and centralised default management. The context can also serialise itself into manifest-friendly payloads when stages record configuration snapshots.
