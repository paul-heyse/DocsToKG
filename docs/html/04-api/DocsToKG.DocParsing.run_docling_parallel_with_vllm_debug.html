1. Module: run_docling_parallel_with_vllm_debug

This reference documents the DocsToKG module DocsToKG.DocParsing.run_docling_parallel_with_vllm_debug.

Legacy PDF → DocTags Converter with vLLM (DEPRECATED).

⚠️ This script is deprecated. Use the unified CLI instead: python -m DocsToKG.DocParsing.cli.doctags_convert –mode pdf

This shim forwards invocations to the unified CLI for backward compatibility and will be removed in a future release.
