1. Module: pdf_pipeline

This reference documents the DocsToKG module DocsToKG.DocParsing.pdf_pipeline.

PDF conversion pipeline with CUDA safety guarantees.

This lightweight module exists primarily so test coverage can assert that the DocsToKG PDF conversion entrypoint enforces the spawn multiprocessing start method. The implementation keeps a close surface to the legacy script without depending on heavyweight optional libraries, allowing tests to monkeypatch behaviour as required.
