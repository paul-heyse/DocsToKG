## Hybrid Search Module Migration Guide

- `DocsToKG.HybridSearch.operations` → `DocsToKG.HybridSearch.service` (pagination & stats) and
  `DocsToKG.HybridSearch.vectorstore` (state serialisation).
- `DocsToKG.HybridSearch.results` → `DocsToKG.HybridSearch.ranking`.
- `DocsToKG.HybridSearch.similarity` → `DocsToKG.HybridSearch.vectorstore`.
- `DocsToKG.HybridSearch.retrieval` → `DocsToKG.HybridSearch.service`.
- `DocsToKG.HybridSearch.schema` → `DocsToKG.HybridSearch.storage`.
- `DocsToKG.HybridSearch.tools.*` → `python -m DocsToKG.HybridSearch.validation`.

Shims emit `DeprecationWarning` throughout the DocsToKG v0.5.x series and will be removed in v0.6.0.
